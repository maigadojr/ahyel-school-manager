# AHYEL School Manager

AHYEL School Manager is a simple school management web application built with HTML, CSS, and JavaScript.

## Features

- Student Registration
- Student Search
- Attendance Management
- Result Management
- Dark Mode
- Export Student Report as PDF
- Responsive Mobile Design
- Local Storage Support

## Technologies Used

- HTML5
- CSS3
- JavaScript
